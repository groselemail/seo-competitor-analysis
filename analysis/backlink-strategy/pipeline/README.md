# Pipeline: Ссылочный анализ

Промежуточные данные ссылочного анализа (Phase 2).

## Структура

| Директория | Содержание | В git? |
|-----------|-----------|--------|
| `intermediate/` | JSON-агрегаты R1-R4 (инвентаризация, языковые ссылки, PBN-сигналы, пересечения, ссылочные профили, валидация, gambling-фильтр, верификация) | Да (~2.8 МБ) |
| `sections/` | Markdown-секции 4 исследований (R1-R4) | Да (~128 КБ) |
| `scripts/` | Python-скрипты (нормализация, pipeline, верификация) | Да (~176 КБ) |
| `normalized/` | Нормализованные TSV-файлы (39 файлов × 12 доменов) | **Нет** (~109 МБ, в .gitignore) |

## Воспроизведение

Для полного воспроизведения анализа необходимы:
1. Сырые CSV-файлы Ahrefs (39 файлов, см. [raw-data/README.md](../../../raw-data/README.md))
2. Python 3 + pandas + numpy
3. Claude Code CLI

Порядок: нормализация (`scripts/run_pipeline.py`) → агрегация → анализ Claude AI → верификация (`scripts/verify_report.py`).

Подробная методология: [08-methodology.md](../../../reports/backlink-strategy/08-methodology.md)
